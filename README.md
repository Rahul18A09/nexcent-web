# Nexcent Landing Page

A modern and responsive **Nexcent Landing Page** built with **Next.js**, focused on creative digital services including **Photography, Editing, Marketing, and Web Solutions**.

This project was developed using **Figma Dev MCP Mode** with **Cursor AI**, transforming design concepts into a clean, production-ready frontend experience.

## ✨ Features

- Modern & Responsive UI
- Photography & Creative Service Showcase
- Marketing & Web Service Sections
- Smooth User Experience
- Clean Component-Based Structure
- Fast & Optimized Performance

## 🛠️ Tech Stack

- **Next.js**
- **React**
- **Tailwind CSS**
- **Framer Motion**
- **Figma Dev MCP**
- **Cursor AI**

## 🚀 Getting Started

Clone the repository and install dependencies:

```bash
npm install
npm run dev
```

Run the development server and open:

```bash
http://localhost:3000
```

## 📌 Development Workflow

**Design → Figma Dev MCP → Cursor AI → Next.js Implementation**

The UI was generated and refined using **Figma Dev MCP Mode** in **Cursor AI**, then customized and optimized for responsiveness and performance.
